# Dashboard

Дашборд на React + TypeScript для просмотра пользователей и постов из Gorest API.

## Быстрый старт

1. Установить зависимости:

```bash
npm install
```

2. Запуск в dev режиме:

```bash
npm run dev
```

3. Запуск для прода:

```bash
npm run build
```

4. Локальный просмотр сборки:

```bash
npm run preview
```
